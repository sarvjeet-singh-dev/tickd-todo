# Tickd Todo Web App

## Overview

Tickd is a task management web application built using a custom SER (State, Events, Render) architecture. The application allows users to efficiently manage daily tasks through task creation, editing, deletion, filtering, searching, priority management, and theme customization while maintaining a clean separation between application state, user interactions, and UI rendering.

## Features

### Task Management

* Create new tasks
* Edit existing tasks
* Delete tasks
* Mark tasks as completed
* Update task information at any time

### Organization & Productivity

* Assign priority levels to tasks
* Search tasks instantly
* Filter tasks by:

  * All Tasks
  * Active Tasks
  * Completed Tasks

### User Experience

* Dark Mode / Light Mode toggle
* Responsive design
* Clean and intuitive user interface

### Data Persistence

* Local Storage integration for persistent task storage across browser sessions

## Architecture

This project follows a custom SER architecture:

### State

Responsible for managing application data and task state.

### Events

Handles user actions and application events.

### Render

Updates the user interface whenever the application state changes.

This approach promotes maintainability, scalability, and predictable UI updates.
